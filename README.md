# 🐶 DogeBot Bybit

Bot criado para identificar oportunidades de entrada no mercado futuro do par **DOGEUSDT** na Bybit.

## 🎯 Objetivo

Detectar automaticamente entradas em operações **long** ou **short** com base em indicadores técnicos no gráfico de 4 horas.

## 📊 Indicadores utilizados

- EMA (médias móveis)
- RSI
- ADX com +DI e -DI
- Padrões de candles

## ⚙️ Funcionalidades

- Sinal de entrada com direção (long/short)
- Recomendação de alavancagem (até 15x)
- Porcentagem da banca sugerida
- Cálculo de Stop Loss e Take Profit
- Alerta visual e sonoro no navegador

## 🚀 Como usar

1. Abrir o arquivo `index.html` no navegador.
2. Esperar a análise.
3. Seguir a recomendação exibida (se houver entrada válida).

## ⚠️ Aviso

Uso experimental e educacional. Não é recomendação de investimento.
